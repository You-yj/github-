# 规则2：

对于字体修饰命令，例如`\mathrm`、`\mathbf`、`\boldsymbol`、`\bm`等，在修饰字体前必须在后面**加空格**

规范形式：`\boldsymbol x`、`\boldsymbol {x}`、`\boldsymbol {x+y}`

不规范形式：`\boldsymbol{x}`、`\boldsymbol{x+y}`

例如：我写下面这段话

- 设 $\boldsymbol x=\boldsymbol x_i+\boldsymbol {y_i+z_i}$，此时该公式为规范形式，为`$\boldsymbol x=\boldsymbol x_i+\boldsymbol {y_i+z_i}$`
- 设 $\boldsymbol{x}=\boldsymbol x_i+\boldsymbol {y_i+z_i}$，此时该公式等号左边部分改为不规范形式，为`$\boldsymbol{x}=\boldsymbol x_i+\boldsymbol {y_i+z_i}$`
- 设 $\boldsymbol x=\boldsymbol x_i+\boldsymbol{y_i+z_i}$，此时该公式等号最后部分改为不规范形式，为`$\boldsymbol x=\boldsymbol x_i+\boldsymbol{y_i+z_i}$`

这时，上传至 github 看看效果对比：

$\lambda_i(\boldsymbol{x}_j)=\delta_{ij}$

$\lambda_i(\boldsymbol{x}_{j})=\delta_{ij}$

$\lambda_i(\boldsymbol{x}j)=\delta_{ij}$
